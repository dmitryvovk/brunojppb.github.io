React para Iniciantes (Portuguese)

Aprenda fundamentos sólidos do React construindo uma aplicação inteiramente do zero.

Se você sempre quis aprender a criar aplicações web modernas e dinâmicas, o React vai te ajudar a ser 10 vezes mais produtivo. Nesse curso, aprenderemos todos os fundamentos sólidos do React.

E o melhor de tudo, durante todas as aulas, você aplicará todos os conhecimentos na criação de uma aplicação feita totalmente do zero.

Ao final do curso, você possuirá uma aplicação completa para colocar em seu portfólio de desenvolvedor e até utilizar o código fonte como base para suas futuras aplicações e entrevistas de trabalho.

Ruby on Rails 5 na Prática (Portuguese)

Aprenda a construir aplicações Ruby on Rails na prática

Quer aprender a criar Aplicações Web com agilidade, utilizando um dos frameworks mais modernos? Este é o curso que você precisa para criar todos os fundamentos necessários para desenvolver Aplicações Web dinâmicas, responsivas e utilizando banco de dados.

Neste curso iremos construir uma aplicação completa, desde a concepção dos modelos em banco dados até o cadastro e autenticação de usuário.

Aqui veremos os princípios do MVC, Validações de formulários, relacionamento entre modelos e o uso do Twitter Bootstrap para criação de frontend resposivos.

No final deste curso, você terá uma aplicação completa, pronta para ser colocada online.

Ruby para iniciantes (Portuguese)

Aprenda a programar com a linguagem Ruby

É um desenvolvedor iniciante e quer aprender a programar com uma das linguagens mais ágeis e modernas? Tem interesse em aprender desenvolvimento Web com Ruby on Rails? Então essa é a base sólida que você precisa para desenvolver aplicações Ruby.

Neste curso você aprenderá os fundamentos da linguagem Ruby, a biblioteca padrão e como utilizar as partes essênciais que são de extrema necessidade para qualquer desenvolvedor.
